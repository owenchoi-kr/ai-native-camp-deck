담인

오담인 SA

고객 기술 문의 분석기 + Amplitude 크롬 확장

Problem

뉴스레터 콘텐츠에서 '아젠다'만 정리하느라 정작 팀에 유용한 인사이트를 놓침. Amplitude 데이터 확인에 여러 단계가 필요

Solution

기술 문의를 자동 분류·분석하는 도구 + Amplitude 데이터를 즉시 조회하는 크롬 확장 프로그램 개발

"뉴스레터에서 '아젠다'가 아닌 '인사이트'를 뽑기 시작했습니다"

— 실무 즉시 적용

서한솔 Sales

초청리스트 자동화 — invite-list 스킬

Problem

Solution

전체 파이프라인을 하나의 Claude Code 스킬로 자동화. 캠프 이후에도 응용 버전으로 실무 적용 중

영주

오영주 PW

PW 가이드 80편 AI 최적화

Problem

80편의 제품 가이드를 일일이 수동으로 최적화하기엔 시간과 인력이 부족. 도구나 하드스킬 없이는 엄두를 못 내던 작업

Solution

Claude Code로 가이드 분석 → 개선점 도출 → 자동 리라이팅 파이프라인 구축

"굉장히 empowering한 경험이었어요.
도구 경험도 하드스킬도 없어서 어찌할 도리가 없던 일에
방법을 찾고 구현할 수 있다는 게 너무 놀랍고 신나요"

— 오영주
